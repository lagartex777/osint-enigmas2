# OSINT ENIGMAS

Este repositório contém um site estático com 31 desafios OSINT de treino. (Mais desafios embreve]

## Conteúdo
- `index.html` = página principal.
- `styles.css` = estilos.
- `script.js` = lógica e UI.
- `challenges.json` = dataset com 31 desafios (contém respostas-format de flags para teste).
- `README.md` = instruções.

## Observações de segurança e ética
Estes desafios são simulados e servem para treinamento. **Não use técnicas de OSINT para invadir contas privadas, roubar dados ou cometer crimes.** Respeite termos de serviço, leis e privacidade.

## Personalização
- Para adicionar/modificar desafios, edite `challenges.json`.
